# customer-behavior-analysis
Data analysis project to explore customer purchasing behavior and generate insights
# Overview

This project analyzes customer purchasing behavior to identify patterns and extract actionable insights.

# Key Insights
Customers aged 20–40 form the core segment.
Spending patterns are consistent across genders.
Income alone is not a reliable predictor of spending.
Distinct customer segments indicate opportunities for targeted strategies.
#Tools Used

Python, Pandas, Matplotlib, Seaborn

#Conclusion

The analysis highlights the value of simple, data-driven approaches in understanding customer behavior and improving decision-making.
